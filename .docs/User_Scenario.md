# 마이링크 (MyLink) 사용자 시나리오

## 1. 방문자 (Visitor): 마이링크 페이지 방문 과정

- **접속**: 방문자는 특정 크리에이터나 개발자의 다양한 작업물을 한곳에서 확인하기 위해 공유받은 마이링크 URL(`domain.com/displayName`)을 클릭한다.
- **정보 확인**: 방문자는 이 계정이 제공하는 콘텐츠의 성격을 파악하기 위해 화면 상단에 렌더링된 실제 이름(`username`), 닉네임(`displayName`), 소개글(Bio)을 읽는다.
- **링크 탐색**: 방문자는 자신이 원하는 특정 사이트(예: GitHub, 블로그 등)를 찾기 위해 사이트별 파비콘이 붙어있는 링크 리스트들을 살펴본다.
- **이동**: 방문자는 목적지로 이동하기 위해 목록에 있는 특정 링크를 클릭한다.

---

## 2. 소유자 (Owner): 링크 추가/수정/삭제 과정

### 2.1 링크 추가 (Add)
- **추가 준비**: 소유자는 새로운 소셜 계정이나 이력서 링크를 등록하기 위해 대시보드 화면에 있는 '새 링크 추가' 버튼을 클릭한다.
- **정보 입력**: 소유자는 방문자가 링크를 알아볼 수 있도록 노출된 폼(Form)에 링크 제목(Title)과 URL 주소를 직접 타이핑하거나 붙여넣기 한다.
- **등록 완료**: 소유자는 입력을 마치고 서비스에 등록하기 위해 '추가(완료)' 버튼을 클릭한다. (시스템은 내부적으로 타겟 URL의 파비콘을 즉각적으로 가져와 리스트 아이콘으로 적용한다.)

### 2.2 링크 수정 (Edit) - 인라인 편집
- **수정 진입**: 소유자는 잘못 입력했거나 업데이트가 필요한 링크 정보를 수정하기 위해 대시보드 리스트의 텍스트(제목 혹은 URL) 영역을 직접 클릭한다.
- **인라인 편집**: 소유자는 수정 모드(입력 폼 상태)로 자연스럽게 전환된 영역에 변경할 텍스트를 타이핑한다.
- **수정 완료**: 소유자는 수정 사항을 데이터베이스에 반영하기 위해 Enter 키를 누르거나 입력란 바깥 화면(여백)을 클릭(블러 아웃)한다.

### 2.3 링크 삭제 (Delete)
- **삭제 요청**: 소유자는 더 이상 공유를 원하지 않거나 유효하지 않은 링크를 지우기 위해 해당 링크 항목 우측의 '삭제(또는 휴지통 아이콘)' 버튼을 클릭한다.
- **삭제 확정**: 소유자는 실수로 삭제하는 것을 방지하고 최종 파기를 확인하기 위해 시스템이 띄우는 알림창(Confirm Alert)에서 '삭제 진행'을 누른다.
